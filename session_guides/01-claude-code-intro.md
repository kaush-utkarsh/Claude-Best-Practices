## Key Commands to Show Live

| What you type | What it does |
|---|---|
| `claude` | Opens Claude Code in current directory |
| `claude "explain this file"` | Quick one-shot question |
| `/help` | Shows all slash commands |
| `/add file.js` | Adds a file to context |
| `/clear` | Clears conversation history |
| `Escape` | Interrupts Claude mid-response |
