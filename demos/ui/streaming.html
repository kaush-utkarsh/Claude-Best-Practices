One flag. Massive UX difference.
Streaming: stream: true in your API call + ~40 lines to read the SSE response.
The user sees the first token in <500ms instead of waiting 6-8 seconds for the full response.
This is the single biggest UX improvement a frontend dev can ship in one day.
